Tutorial Git Untuk Pemula (Indonesia)

Written by Raibima Putra

Halo! Di blog post pertama saya ini, saya akan coba jelaskan sedikit mengenai Git Version Control System (VCS).

Apa itu git?

Git, singkat cerita, adalah sebuah Version Control System (VCS), yaitu sebuah tool yang dapat digunakan untuk memantau perubahan-perubahan yang diterapkan ke suatu file atau folder.

Contoh penggunaan Git

Pastikan Git telah terinstal di komputer anda.

  • Pengguna Windows: Jalankan Git Bash.
  • Pengguna Mac/Linux: Jalankan terminal.

Buatlah suatu folder baru, namanya bebas, dan masuk ke dalam folder tersebut:

$ mkdir folder-baru
$ cd folder-baru

Beri tahu Git bahwa anda akan bekerja di folder ini:

$ git init

Sekarang anda bisa mulai bekerja di dalam folder ini. Sebagai contoh, buatlah file sederhana dan beri tahu git bahwa anda ingin file tersebut dipantau perubahan-perubahannya:

$ touch contoh-file.txt
$ git add contoh-file.txt

Ingat bahwa Git adalah Version Control System (VCS)! Anda perlu membuat versi untuk setiap perubahan yang anda terapkan dan memberikan pesan mengenai perubahan-perubahan apa saja yang anda telah terapkan pada file-file yang sedang anda kerjakan:

$ git commit -m "Commit pertama. Membuat file kosong bernama contoh-file.txt :)"

Jika sudah, anda bisa coba membuat perubahan-perubahan pada file yang baru saja diberikan versi. Silakan edit file contoh-file.txt, tambahkan konten apa saja, lalu simpan. Anda baru saja merubah sesuatu di file tersebut, mari kita lihat perubahan apa saja yang sudah anda lakukan:

$ git diff

Seperti yang anda lihat, Git sangatlah pintar. Git tahu apa saja yang anda ubah sejak anda terakhir menetapkan suatu versi. Sama seperti tadi, perubahan yang anda terapkan harus diberi versi baru:

$ git add contoh-file.txt
$ git commit -m "Sedikit perubahan di dalam contoh-file.txt."

Kok repot sekali, ya? Masa sih tiap kali ada file berubah harus kita ‘add’ lalu ‘commit’? Jangan salah, Git ternyata punya jalan pintas untuk melakukan dua hal tersebut sekaligus dalam satu perintah saja:

$ git commit -am "Sedikit perubahan di dalam contoh-file.txt."
